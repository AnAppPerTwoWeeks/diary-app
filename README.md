# 짧은 일기

짧은 일기는 하루를 사진과 함께 간단하게 기록할 수 있는 앱 입니다.

## 앱 스토어

https://apps.apple.com/app/짧은-일기/id1502528895

## 스크린샷

![](screenshot.png)

## 주요기능

- 기본적으로 카드 형식의 일기가 보입니다
- 리스트로 볼 수도 있어요
- 사진을 추가하고 간단한 내용을 입력하면 끝!
- Face ID / Touch ID를 사용해 앱의 접근을 제한할 수 있어요
